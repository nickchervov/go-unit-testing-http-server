# 🛒 Shopping List API - RESTful API в Go

> Современный, хорошо протестированный REST API для управления списком покупок на чистом Go с использованием chi router и comprehensive unit-тестов.

## ✨ Особенности

- 🚀 **RESTful архитектура** — полный CRUD цикл для товаров
- ✅ **Полное покрытие тестами** — все обработчики протестированы с edge cases
- 🔒 **Валидация данных** — строгая проверка входных параметров
- 📊 **Фильтрация по категориям** — поиск товаров по категории
- 🎯 **Чистый код** — разделение ответственности (models, handlers)

## 🛠️ Технологический стек

| Компонент | Технология |
|-----------|------------|
| Язык | [Go](https://go.dev/) 1.21+ |
| Router | [chi/v5](https://github.com/go-chi/chi) |
| Тестирование | `testing`, [testify](https://github.com/stretchr/testify) |

## 📡 API Endpoints

| Метод | Endpoint | Описание |
|-------|----------|----------|
| `GET` | `/items` | Получить все товары |
| `POST` | `/items` | Добавить новый товар |
| `GET` | `/items/{id}` | Получить товар по ID |
| `PUT` | `/items/{id}` | Полное обновление товара |
| `PATCH` | `/items/{id}` | Частичное обновление (только изменённые поля) |
| `DELETE` | `/items/{id}` | Удалить товар |
| `GET` | `/items/category/{category}` | Получить товары по категории |


## 🧪 Тестирование

Проект имеет **полное покрытие unit-тестами**:

- ✅ Все HTTP обработчики протестированы
- ✅ Edge cases и валидация покрыты
